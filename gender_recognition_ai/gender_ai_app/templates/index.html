About the Project

Gender Recognition AI's neural network is made with a Keras sequential model. The model consists of three dense layers: the input, hidden, and output layers. Before a photo can be passed through the gender recognition neural network, it is passed through a facial recognition neural network, supplied by the Face Recognition library, which always outputs a length 128 vector containing information on the outlines of various facial features.

This 128 length vector is then passed into the input layer of the gender recognition neural network, which has 128 neurons. The hidden layer contains 65 neurons and the activation functions is ReLU. The output layer is a single neuron with the sigmoid activation function such that it outputs a value between 0 and 1 for the probability that a given face belongs to a male. Please visit the GitHub repository for a more detailed look into how the model was coded.
